## 小型化オプション

### 現在のサイズ: 基板55×55mm / 筐体60×60×25mm

### Option 1: 基板を45×45mmに縮小 (+0円)
- 現在の実装率21%なので余裕あり
- EKMC1603111 (TO-5 D9.9mm) が最大の制約だが端に配置で対応
- **筐体: 50×50×22mm** に縮小可能
- 壁面設置時の存在感がさらに薄くなる

### Option 2: EKMC1603111 → AMN43111 (SMD PIR) への換装
- AMN43111: SOT-363風フラットパッケージ (5×5mm) vs TO-5 (D9.9mm)
- 高さ: 4mm vs 8.6mm → **基板高さを大幅削減**
- 価格: $3.20 ($4.80比 -$1.60の削減も兼ねる)
- **筐体高さ: 25mm → 16mm** に可能

```
現行筐体:     60×60×25mm
Option 1:    50×50×22mm (-基板縮小のみ)
Option 2:    50×50×16mm (-基板縮小 + PIR SMD化)
超小型版:    45×45×14mm (全て最適化した場合)
```

### Option 3: ESP32-C6-MINI-1 への変更
- ESP32-C6: WiFi 6 + BLE 5.3 + **Thread 1.3 + Zigbee 3.0** 追加対応
- 価格: $2.80 (ESP32-S3比 -$0.40)
- Matter対応によりApple Home / Google Home との統合が容易に
- ⚠️ GPIOピン数が若干少ない → 設計確認必要

## 防水化オプション

### IP等級と用途の対応

| 等級 | 内容 | 追加コスト | 推奨用途 |
|------|------|-----------|---------|
| IP20 | 現状 (保護なし) | $0 | 室内専用 |
| **IP44** | **水の飛沫保護** | **+$0.50** | **民泊室内 (推奨)** |
| IP65 | 防塵+水の噴流 | +$2.00 | 屋外軒下 |
| IP67 | 防塵+水中1m/30分 | +$4.00 | 屋外完全防水 |

### IP44達成方法 (推奨 / +$0.50)

1. **基板へのコーティング**: アクリル系コンフォーマルコート (HumiSeal 1A33)
   - スプレー1本 $15 → 約200台分
   - 全部品の上からスプレー → 結露・湿気から保護
   - USB-CとJSTコネクタはマスキングしてから塗布

2. **筐体にゴムガスケット追加**:
   - 前面カバーと本体の接合部に1mmシリコーンパッキン
   - JLCPCB筐体発注時に溝加工を追加指定
   - PIR窓: フレネルレンズとの接着にシリコーン系接着剤使用

3. **USB-C開口部**: ゴムキャップ同梱 (使用時は外す)

### IP65達成方法 (+$2.00)

- 筐体を完全密閉設計に変更
- **VOCセンサー(SGP40)に問題**: 外気との通気が必要なため防水と相反
  - 解決策: 通気性防水膜 (Gore-Tex テープ, ~$0.30) を通気スリットに貼付
  - 膜で水は防ぎつつガス分子は通過させる
- マイク(INMP441)の音孔: 防水メッシュ (IP67対応品, ~$0.20) に変更
- PIR窓: シリコーンシールで完全封止

### 屋外設置版の追加検討事項
- 動作温度保証: -20°C〜+70°C (現行部品の多くは0〜60°C保証)
  - SGP40: -40〜85°C ✅
  - SHT40: -40〜125°C ✅
  - BMP280: -40〜85°C ✅
  - ESP32-S3: -40〜105°C ✅
  - ME6206A LDO: -40〜125°C ✅ (問題なし)

## 機能追加オプション (将来版)

### CO₂センサー追加 → Proモデルの差別化要因
- **SGP40はVOC Indexのみ**: 実際のCO₂濃度は測定しない
- **SCD41** (Sensirion, $14): 真のCO₂測定 (400-5000ppm, ±40ppm)
- 民泊での喚起アラートの精度が大幅に向上
- **Proモデルのみに搭載** → 価格差 ¥8,800 (Lite) / ¥14,800 (Hub) / **¥24,800 (Pro + CO₂)**

### 太陽電池対応 (屋外版)
- 小型ソーラーパネル (55×55mm, 1W) を筐体上面に追加
- TP4056充電IC ($0.10) + 350mAh LiPoバッテリー ($1.50)
- USB-Cは充電/ファームウェア書き込み専用に変更
- 完全ワイヤレス設置が可能に (民泊の配線問題を解決)
- 追加コスト: +$2.50/台

### Matter / HomeKit 対応
- ESP32-C6に変更するだけでMatter 1.3対応可能
- Apple HomeKit, Google Home, Amazon Alexaから直接アクセス
- KAGIアプリと並行してホームオートメーションにも対応
- 開発工数: ファームウェアに`esp-matter` SDK追加 (Espressif公式)

### e-ink ディスプレイ追加
- 29mm×13mm e-ink (GDEW0154M10, $2.50)
- 現在の温度・湿度・空気質を筐体表面に常時表示
- 消費電流: 更新時のみ (平均1μA以下)
- 民泊ゲストが一目で室内環境を確認できる
