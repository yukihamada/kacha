# Airbnb Community Japan 投稿

**カテゴリ**: ホスティングのヒント / ツールとテクノロジー
**タイトル**: スマートロック x Beds24の自動連携アプリを作りました

---

こんにちは。都内で3年ほどホストをしています。

ホストを始めた当初から悩んでいたのが、**鍵の受け渡し**と**多言語でのゲスト対応**です。

最初はキーボックスを使っていましたが、暗証番号の変更忘れが怖い。SwitchBotロックに変えたら便利になったけど、今度はBeds24との連携が手動。チェックイン/アウトの度にSwitchBotアプリとBeds24を行き来して、ゲストには毎回手打ちでドアコードを送る生活でした。

特に辛かったのが深夜の対応です。時差のある海外ゲストから「エントランスの開け方がわかりません」と午前3時にメッセージが来る。しかも英語や中国語で。

---

## 作ったもの

この問題を解決するために、**KAGI（カギ）**というiOSアプリを作りました。

やっていることはシンプルです:

1. **Beds24の予約を自動で取得**して、カレンダーに表示
2. **スマートロック（SwitchBot / Sesame / Nuki / Qrio）をワンタップで操作**
3. **ゲストのメッセージにAIが返信候補を提案**（多言語対応）
4. **ゲストカード**（WiFi、ドアコード、ハウスルール）を暗号化リンクで共有

Beds24を使っている方なら、Invite Codeを入力するだけで予約が同期されます。

---

## 使い方の流れ

設定は10分くらいです:

1. アプリをインストール
2. Beds24のInvite Codeを入力（Beds24の設定画面 → APIから発行）
3. スマートロックのアカウントを連携
4. 各物件にどのロックを使うか設定

あとは自動です。新しい予約が入れば通知が来るし、ゲストからメッセージが来ればAIが返信候補を出してくれます。
